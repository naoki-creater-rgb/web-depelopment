イベント調整API テストコンソール

接続先: http://localhost:8000 ／ 初期データが投入済みです（manager_1, guest_1〜3, event 1・2）

POST アカウント作成

/create/new_account
同じIDで2回送ると重複エラーになります。

POST ログイン

/login

GET 幹事: 管理イベント一覧

/manager/events

GET 参加者: 参加イベント一覧

/participant/events
guest_1=回答済 / guest_2=未回答 で違いが見られます。

GET イベント詳細

/event/detail
event 1・2 が初期投入済み。確定済みなら confirmedDetails が返ります。

POST 幹事: イベント作成

/create/new_event

POST 参加者: 回答送信

/participant/create/answer

POST 幹事: イベント確定

/confirm_information
レスポンスがここに表示されます
